2. Evaluate complexity factors: files affected, cross-cutting concerns, new patterns, security sensitivity, data model changes, external dependencies, reversibility
3. Compute complexity score (0-100%)
4. Select pathway: TRIVIAL (0-20%), STANDARD (21-55%), COMPLEX (56-100%), INFRASTRUCTURE
5. Output the governance assessment with active phases and depth levels
6. Load applicable rules from `.ai-governance/rules/_index.yaml`
7. Wait for human approval before proceeding
